# Conservation

## Overview

Manage the Kakapo conservation effort!
Start with an isolated single island and go to returning them to the mainland

## Concepts

Different islands support a different population level, but could be more isolated and take more resources

They can have different weather patterns, tree population, predator levels etc

Different terrain difficulty so management is different

Mainland resources like vets, universities, even foreign resources (tech/academic)

Lobby politicians for more funding

Manage predator free policy etc to get islands cleared

Manage infections/diseases/misc health issues

Genetic profiling fix inbreeding/genetic bottleneck etc

Public awareness/donations
